# Expense_Tracker_CLI

A simple Command Line Expense Tracker built using Java.
This application allows users to record, view, and manage daily expenses directly from the terminal.

The project demonstrates core Java concepts such as:

Object-Oriented Programming (OOP)

File handling (CSV storage)

Collections

CLI menu design

Data parsing

🚀 Features

➕ Add new expense

📋 View all expenses

🗑 Delete an expense

📊 Calculate total expenses

💾 Persistent storage using CSV file

📅 Uses LocalDate for date handling

🛠 Tech Stack

Language: Java

Storage: CSV file (expenses.csv)

Concepts Used:

OOP

File I/O

Lists (ArrayList)

Exception Handling

Java Time API
